# The Technical Logic & Systems Toolkit 

This file contains a set of 10 analytical prompts designed to stress test solutions and uncover blind spots in your reasoning.

Primary Uses: Root cause analysis (identifying meta problems), strategic foresight (second order consequences, pre mortems), and decision engineering (matrixes, MVPs, and reverse engineering).

Goal: To move beyond surface level fixes and apply rigorous structural thinking to ensure you are solving the right problems efficiently.
